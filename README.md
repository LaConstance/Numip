# Numip - Numbers in Picture

A minimalist, distraction-free counter that floats over your work.  
Built to track exercises while keeping your study materials visible with Picture‑in‑Picture.

## ✨ Features

- **Picture‑in‑Picture** – Keep the counter always on top of other windows, even during presentations or full‑screen apps.
- **Double‑click to edit** – Correct a value instantly without repetitive clicking.
- **Dark & light mode** – Respects your system, with smooth transitions and persistent preference.
- **Reset button** – One click to start fresh.
- **Persistent count** – Automatically saved in your browser. Never lose your progress.
- **Responsive design** – Uses `vmin` units; looks great on desktop and mobile.
- **Zero dependencies** – Pure HTML, CSS, and vanilla JavaScript.

## 🚀 Live Demo

Try it online: [Numip](https://laconstance.github.io/Numip/)

## 📦 Usage

1. Click **+** or **−** to adjust the counter.
2. Double‑click the number to type an exact value.
3. Use the **PiP button** (top‑right) to pop the counter into a floating window.
4. Switch theme with the **sun/moon button** (top‑left).
5. Hit **↻** to reset.

## 🛠 Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, `vmin`)
- Vanilla JavaScript (Document Picture‑in‑Picture API, `localStorage`)
